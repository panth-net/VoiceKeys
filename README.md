# VoiceKeys
Voice dictation on your computer via Deepgram API

## Setup:
0. Create your API key here: https://deepgram.com/ (Click Advanced when creating the API key and select Admin if you'd like to see your usage within the application)
1. Open the application (it will appear in your Taskbar (In Windows, it will appear in the System tray and if you'd like, you can drag and drop it into your Taskbar)
2. Open the application and paste your Deepgram key into **Deepgram API Key**
3. Click **Save API key**
4. Change the hotkeys to start and stop the recording. Remember, you must first enable all permissions before the hotkeys will work.

## Installing an updated version
On Mac, please make sure to **erase** (not just toggle off, you must remove) the VoiceKeys application from each of the following paths in your Mac Settings. Then install the new VoiceKeys application. Then add these settings back:
   - System Settings > Privacy & Security > Microphone
   - System Settings > Privacy & Security > Accessibility
   - System Settings > Privacy & Security > Input Monitoring

Then restart the application.
